🔊 整活神器 · GitHub Copilot Agent Skill

promote  ≠  prompt

一个帮你分清 prompt 和 promote 的 Copilot Agent Skill

⭐ 在 GitHub 查看 & Star

📖 发音对比

promote
/prəˈmoʊt/
❌ 你以为你在说的
prompt
/prɒmpt/
✅ 你其实想说的

语音由 Ming-omni-tts-16.8B-A3B 生成， 你可以在 ling-open-studio 生成语音。

⚡ 安装方法

1

将以下这句话发给你的 GitHub Copilot Agent，即可完成安装：

请从 https://github.com/cafe3310/promote-skill 安装 promote skill，将 .github/agents/promote.md 复制到我的仓库中。
2

或者直接克隆并手动复制 skill 文件：

cp .github/agents/promote.md /your-repo/.github/agents/promote.md
3

安装完成后，跟你的 Copilot 说一句 "promote 怎么读"，静候奇迹发生 🎉

🛠️ 工作原理

Skill 激活后，自动检测操作系统，调用最直接的 TTS 命令朗读纠错内容：

🍎
macOS say -v Ting-Ting "我注意你刚才说了 promote…"
🐧
Linux espeak-ng -v zh "…" / espeak "…" (fallback)
🪟
Windows PowerShell SAPI SpeechSynthesizer
